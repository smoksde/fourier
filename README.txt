This repository contains C++ implementations of the fast fourier transform algorithm and the inverse fast fourier transform algorithm.

CMake is used as a cross-platform build system.
You should install CMake to be able to build and compile.

Compiling:
    1. Open the console
    2. Navigate to the build directory
    3. Type "cmake .."
    4. Type "make"
    5. Run the executable with "./fourier"
